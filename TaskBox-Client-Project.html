Working Lojic

It is an software to inform the material preparation that it is in the raw material / part keeper of the bench and the materials brought to the benches with matarial cases in the production line. It is aimed to develop the WCM.

Sensors on 8 different looms receive the data from the Profinet connection on PLCs on the charcoal and transmit the data to the main Siemens S7-300 Profinet Supported PLC. These addresses that receiving signal from the sensors creates tasks to logistic department. When the signal is given within the time frame of the tasks performed, the box is expected to be collected. If the critical time is exceeded, the task background color will turn red. After the sensor is opened, the task is automatically deleted.

Main menu contains “Profinet I/Q”, ”Seri Port(Serial Port)”, ”Görev Raporu(Task Report)”, ”Tezgah Hareket Bilgileri(Case Movement Informations)”, ”Ürün Ağacı(Product Trees”, ”Personel Tanımları(Personal Informations)”, ”Ayarlar”(Options), ”Çıkış(Exit)” pages. “Oturum Aç(Login)”, "Oturum Durumu(Login Status)", ”Tarih(Current Date)”, ”Vardiya(Turn of work)", ”Saat(Clock)" and “Verim(Effeciency)” page buttons are on the bottom of the menu.

Effeciency

If the task completed in the countdown time, it effects to efficiency by 100% .When the counter shows 00:00,task row becomes red color and efficiency effect goes to the down from %100. Effeciency graph refreshes itself after every 20 seconds. Effeciencies are group by job rotations.

Task Box Client Side

This system created with C# and S7300 PLC that contain profinet modules.

Login

Users have to be log in for the system processes. Users have permissions types and they just do process that be allowed. Task Box block the wrong process in this way. Every users knows to job informations and they can not get out of this circle.

Users can see the status of defined workstations at the right of the page. If station-sensor connection status is connectable,red boxes becomes a white at the "O" line. When the PLC output receiving signal from the sensor, "I" line box that the matched becomes green. After that signal, task occurs on the main page board with "Station Name","Product Name" and "Countdown" informations. When the case box took from the station, task removes .
